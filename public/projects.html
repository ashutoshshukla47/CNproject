PasswordStrengthChecker
Git Repo

Project Description: A Real-Time Password Validation Tool

  • Built a responsive and interactive GUI using Java Swing, allowing users to check password strength instantly while typing.
  • Implemented five key validation checks: minimum length, presence of uppercase letters, lowercase letters, numbers, and special symbols, with real-time visual feedback.
  • Integrated a dynamic progress bar and strength label that categorizes passwords as Too Weak, Weak, Moderate, or Strong based on compliance with security criteria.
  • Added an intuitive toggle button to show or hide password content securely, improving user convenience and experience.
  • Designed with a visually appealing layout, including dynamic resizing and background support for a polished user interface.
